# car-price-prediction
A machine learning project that predicts the selling price of used cars using regression analysis on features like year, fuel type, and transmission.
Project Overview
The objective of this project was to develop a machine learning model capable of accurately predicting the market price of cars based on various technical and regional features. By leveraging historical sales data, this project aims to assist users in understanding vehicle valuation trends.  Tools and Technologies UsedThis project was implemented using Python in a Google Colab environment, utilizing the following libraries for data manipulation, analysis, and modeling:Data Handling: pandas and numpy were used for data loading, cleaning, and mathematical operations (such as log transformation to normalize price distributions).  Visualization: matplotlib and seaborn were employed for data inspection and trend analysis.  Machine Learning: scikit-learn provided the foundational tools, including train_test_split for data partitioning, LinearRegression for predictive modeling, and various metrics (r2_score, mean_absolute_error, mean_squared_error) for performance evaluation.  
Key Tasks PerformedThe project followed a standard machine learning workflow to ensure high-quality predictions:  Data Collection & Cleaning: Loaded the dataset and identified missing values, particularly in the Engine_displacement and Battery columns.  Feature Engineering:Removed outliers using the Interquartile Range (IQR) method to improve model stability.  Applied one-hot encoding using pd.get_dummies to transform categorical variables (make, model, city) into a numerical format suitable for machine learning.  Applied a log transformation to the target variable (Price_Rs) to stabilize variance and handle skewed price data.  Model Training: Split the data into training (80%) and testing (20%) sets and trained a Linear Regression model.  Evaluation: Assessed the model using R-squared ($R^2$), Mean Absolute Error (MAE), and Root Mean Squared Error (RMSE).
